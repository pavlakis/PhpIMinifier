PhpIMinifier is a directory iterator and minifier using the YUI compressor.

Use the example file: PhpIMinifier_Example.php
The example file should have enough information to get you started.

For more information, look at my blog: http://pavlakis.info/php/11-phpiminifier

Hope you enjoy using it.

Feel free to extend it / change it / adapt it to what you require.
